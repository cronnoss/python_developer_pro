1. Standard input: The file containing the text to search.
2. Command line options: A series of switches that control the behavior of grep. These include options such as -i (case insensitive), -e (match on regular expressions), --column (-c), and so on.

1. Make sure you are in the directory where you want to change the file or directory's permissions.

Question: Что делает команда cd?
Answer: Certainly! The command "cd" stands for "change directory." It is used to change the current working directory (i.e., the location where you are currently running your commands) from one directory to another.
For example, if you want to switch to the folder located at "/home/username", you can type the following command into your terminal or command prompt:
```
cd /home/username
```
This will make you enter that specific directory. You can repeat this command as many times as necessary until you reach the desired destination.

Question: Опиши команду mkdir
Answer: To create a directory, you can use the `mkdir` command, which stands for "make directory". Here's an example of how to execute it on a Linux system:

```bash
# Create a new directory called 'my-directory' in the current directory
mkdir my-directory
```

In this example, we are creating a new directory named `my-directory` in the current directory (`cd`) using the `mkdir` command. The first argument is the name of the directory to be created (in this case, `'my-directory'`), followed by any optional arguments that specify the path or location where the directory should be created.
